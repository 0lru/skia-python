# Skia python binding

Python binding to [Skia Graphics Library](https://skia.org/).

- Binding based on [pybind11](https://github.com/pybind/pybind11).

Supported platforms:

- Linux x86_64
- macOS
- Windows x86_64

Currently, the binding is in under active development.
